How we started


- We started this website to share about the effective learning styles, tips and strategies to all types of students.

- We decided to create an easy website for all students from worldwide to learn!

- Our main purpose is as in the online version, the students don’t need that much time to
search and it will be time saving.

- At the start of this journey, we had faced tons of errors and problems but we managed to come through it.

- We started this website with 4 people: Thiri Shoon Lei, Min Thu Tah, May Myat Thida Kyaw, May Phu Wathone Wai.

Our members

Our group members include Thiri Shoon Lei as the main coder and co designer, Min Thu Tah as the main coder, May Phu Wathone Wai as the idk and May Myat Thida as the information collectro and the designer.

The reason we created this website.

We created this website to help students that are struggling to keep up with their studies and how they can study more effectively so they can get higher marks

How we created this website

We spent about 1 month to create this website, we used Vs code live share to collaborate and code together and we encountered many errors while coding but we managed to fix them.

Errors we encountered

We encountered a lot of errors while creating this website with some taking over 3 days to fix and other minor ones such as the navigation bar not in the right position and some that we cannot fix like the footer wave animation that we ended up deleting it
